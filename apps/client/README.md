## Installation

1. Clone the project:

```bash
git clone https://github.com/Hai1205/MindLog.git
cd MindLog
```

2. Install the packages:

```bash
npm install
```

3. Run the application in development mode:

```bash
npm run dev
```

4. Build the application for production:

```bash
npm run build
```
